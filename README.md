#Cronograma PRO 

Um organizador de estudos desenvolvido em Python para auxiliar na gestão de disciplinas e horários semanais!

#Funcionalidades:
- Cadastro de Disciplinas: Insere matéria, dia e horário;
- Validação Inteligente: Impede campos vazios ou seletores não preenchidos;
- Prevenção de Conflitos: O sistema avisa se você tentar cadastrar duas matérias no mesmo horário;
- Ordenação Automática: Organiza a tabela visual seguindo a ordem lógica da semana (Segunda a Domingo);
- Exportação: Salva seu cronograma em um arquivo ".txt" formatado.

#Tecnologias Utilizadas:
- Python; 
- Tkinter / TTK: Para a interface gráfica moderna (Tema Clam);
- FileDialog: Para manipulação de arquivos no sistema operacional.

#Passo a passo para execução do programa: 
1. Certifique-se de ter o Python instalado;
2. Execute o arquivo principal:
   bash
   python nome_do_seu_arquivo.py
Ou clique em "run".
